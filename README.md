# CS-300 Course Advisor

**Overview**

This project was developed in CS-300: Data Structures and Algorithms.  
The application loads course information from a data file and allows users to view a list of courses and their prerequisites.

**Features**

• Load course data from a file  
• Display all courses in sorted order  
• Display detailed course information and prerequisites  
• Store course data using appropriate data structures

**Concepts Demonstrated**

• Data structure selection  
• Searching and sorting algorithms  
• File input processing  
• Program efficiency considerations
